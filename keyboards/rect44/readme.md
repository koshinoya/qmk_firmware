# rect44

![rect44](https://github.com/koshinoya/Keyboards/blob/master/rect44/img/rect44_01.jpg)

A 44~48 keys row staggered keyboard.

* Keyboard Maintainer: [koshinoya](https://github.com/koshinoya)
* Hardware Supported: Pro Micro or [BLE Micro Pro](https://booth.pm/ja/items/1177319)
* Hardware Availability: https://koshinoya.booth.pm/items/1856336

Make example for this keyboard (after setting up your build environment):

    make rect44:default

See the [build environment setup](https://docs.qmk.fm/#/getting_started_build_tools) and the [make instructions](https://docs.qmk.fm/#/getting_started_make_guide) for more information. Brand new to QMK? Start with our [Complete Newbs Guide](https://docs.qmk.fm/#/newbs).

A build guide for this keyboard can be found here: [Rect44 Build Guides(Japanes only)](https://github.com/koshinoya/Keyboards/blob/master/rect44/Rect44_build_guide_jp.md)
